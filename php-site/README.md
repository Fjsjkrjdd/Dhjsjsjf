# Сайт психолога Чернова Н. В. — PHP-версия для хостинга (SpaceWeb и др.)

Готовый сайт на **чистом PHP** (без фреймворков) с собственной **админ-панелью / CMS**,
галереей дипломов, формой записи и подключаемой **онлайн-кассой ЮKassa** (с чеками 54-ФЗ).
Работает на обычном PHP-хостинге (SpaceWeb / sweb, Beget, Reg.ru, Timeweb и т. п.).

- **PHP 7.4+ / 8.x**
- **База данных: SQLite по умолчанию** (ничего настраивать не нужно — файл создаётся сам).
  При желании можно переключить на **MySQL** в `config.php`.
- Расширения: `pdo_sqlite` (или `pdo_mysql`), `curl` (для онлайн-кассы), `mbstring`.

## Установка на хостинг (SpaceWeb)

1. Загрузите содержимое архива в корневую папку сайта (обычно `public_html` или `www`)
   через файловый менеджер панели хостинга или по FTP.
2. Убедитесь, что папки `data/` и `uploads/` доступны для записи (права `755` или `775`).
   Обычно это уже так по умолчанию.
3. Откройте сайт в браузере — база данных и стартовые данные создадутся автоматически
   при первом открытии.
4. Войдите в админку: `https://ваш-домен/admin/`
   - **E-mail:** `admin@chernova-psy.ru`
   - **Пароль:** `admin12345`
5. **Сразу смените пароль** в разделе «Мой профиль» и впишите свой секрет в `config.php`
   (параметр `app_secret`).

> Если сайт устанавливается в подпапку (например, `site/`) — всё работает без изменений,
> ссылки строятся автоматически.

## Использование MySQL (по желанию)

Откройте `config.php` и задайте:

```php
'db_driver' => 'mysql',
'mysql' => [
    'host' => 'localhost',
    'name' => 'имя_базы',
    'user' => 'пользователь',
    'pass' => 'пароль',
    'charset' => 'utf8mb4',
],
```

Таблицы создадутся автоматически при первом открытии сайта.

## Что умеет админ-панель

- **Тексты страниц** — редактирование всех заголовков и текстов (главная, услуги, обо мне и др.).
- **Услуги и цены** — добавление/изменение/удаление, цена, длительность, описание, фото.
- **Дипломы** — загрузка изображений (на сайте — лента с прокруткой и увеличением по клику).
- **Образование**, **Отзывы**, **Статьи** (блог), произвольные **Страницы**.
- **Заявки и оплаты** — все заявки с сайта и статусы платежей.
- **Настройки** — контакты, соцсети, фото, карта, SEO, **виджет отзывов Яндекс**, **онлайн-касса**.
- **Профиль** — смена пароля.

## Отзывы с Яндекс Карт (виджет)

В админке: **Настройки → «Отзывы с Яндекс Карт (виджет)»** — вставьте HTML-код виджета
(из Яндекс Бизнес / конструктора виджетов). Тогда на странице «Отзывы» и на главной
будет отображаться официальный виджет с отзывами. Если поле пустое — показываются отзывы,
добавленные вручную в разделе «Отзывы».

## Онлайн-касса (ЮKassa)

1. Получите в личном кабинете ЮKassa `shopId` и секретный ключ.
2. Админка → **Настройки → Онлайн-касса**: включите оплату, введите данные;
   при необходимости включите чеки (54-ФЗ) и выберите систему налогообложения и ставку НДС.
3. В ЮKassa укажите URL для уведомлений: `https://ваш-домен/pay/webhook`

После этого на странице записи появится опция оплаты картой онлайн.

## Что заполнить владелице сайта

Стартовые данные взяты из профиля на Яндекс Картах. Через админку:
- загрузите **главное фото** и **дипломы** (есть в профиле ВКонтакте / Яндекс);
- уточните **образование** (сейчас заполнители);
- замените **демонстрационные отзывы** или подключите **виджет Яндекса**;
- проверьте контакты и ссылки на соцсети.

## Структура

```
/                  корень сайта
├─ index.php          точка входа (роутинг ЧПУ через .htaccess)
├─ config.php         настройки (БД, секрет) — отредактируйте
├─ .htaccess          ЧПУ + защита служебных файлов
├─ admin/             админ-панель (вход: /admin/)
├─ includes/          ядро: БД, функции, авторизация, ЮKassa
├─ templates/         шаблоны публичных страниц
├─ assets/            CSS и JS
├─ uploads/           загруженные изображения (нужны права на запись)
└─ data/              база SQLite (создаётся автоматически; нужны права на запись)
```

## Локальная проверка (для разработчика)

```bash
php -S localhost:8000 router.php
```
Файл `router.php` нужен только для встроенного сервера PHP; на реальном хостинге
маршрутизацию выполняет `.htaccess`.

## Безопасность

- Смените `app_secret` в `config.php` и пароль администратора.
- Папки `data/` и `includes/` закрыты от прямого доступа через `.htaccess`.
- Загрузка файлов ограничена изображениями.
